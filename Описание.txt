Проект "Соцсеть" (в порядке приоритета)
1) Регистрация/аутентификация
2) Страница с информацией о пользователе
3) Поиск по пользователям
4) Добавление в друзья
5) Диалог с одним\несколькими собеседниками
6) Хранение файлов
7) Отправка файлов в сообщении
8) Функция "ответить" в диалоге
9) Обновление в реальном времени

Routes (authenticated):
1. / - редирект на /user/<id пользователя>
2. /user/:id - страница пользователя с данным id
3. /search - поиск пользователей
4. /edit - изменение личных данных
5. /friends - друзья
6. /dialogs - диалоги 
7. /files - файлы

Routes (unauthenticated):
1. / - страница аутентификации\регистрации
2. Остальные пути - редирект на /
